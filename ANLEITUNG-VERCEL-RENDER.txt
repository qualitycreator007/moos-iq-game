MOOS IQ GAME – EINFACHE ANLEITUNG FÜR VERCEL + RENDER

DU BRAUCHST:
- einen kostenlosen Account bei https://vercel.com
- einen kostenlosen Account bei https://render.com

TEIL 1: BACKEND AUF RENDER
1. Entpacke diese ZIP-Datei.
2. Gehe zu Render.
3. Klicke auf New + und dann Web Service.
4. Lade den Ordner backend hoch oder verbinde ein Git-Repository.
5. Falls Render nach Einstellungen fragt:
   - Root Directory: backend
   - Build Command: npm install
   - Start Command: npm start
6. Veröffentliche den Service.
7. Kopiere dir danach die Internetadresse von Render.
   Beispiel:
   https://moos-iq-backend.onrender.com

TEIL 2: FRONTEND AUF VERCEL
1. Gehe zu Vercel.
2. Klicke auf Add New... und dann Project.
3. Lade den Ordner frontend hoch oder verbinde ein Git-Repository.
4. Warte, bis Vercel die Seite gebaut hat.

TEIL 3: BACKEND-ADRESSE EINTRAGEN
1. Öffne im frontend-Ordner die Datei:
   src/main.jsx
2. Suche nach:
   https://DEINE-RENDER-URL.onrender.com
3. Ersetze das mit deiner echten Render-Adresse.
4. Dasselbe kannst du auch in frontend/index.html prüfen.
5. Lade danach das Frontend neu auf Vercel hoch.

FERTIG
Danach läuft:
- die Webseite auf Vercel
- der Spiel-Server auf Render

WICHTIG
- Render im kostenlosen Tarif kann nach Inaktivität kurz schlafen.
- Der erste Start kann dann ein paar Sekunden dauern.
