# Moos IQ Game Multiplayer Web-App 1.2 Merged

Zusammengeführt:
- 1.1 Spieloberfläche
- Timer-Logik
- Joker-Logik
- 250 Fragen pro Thema
- keine doppelte Frage pro Spielrunde
- Spiel-Highscores
- Konto-Highscores mit addierten Moos Dollars
- PWA- / Standalone-Vorbereitung

Kostenloses Hosting:
- Frontend: Vercel oder Netlify
- Backend: Render oder Railway (Free Tier abhängig von Verfügbarkeit)
- Alternativ alles selbst lokal starten

Start:
Backend:
cd backend
npm install
npm start

Frontend:
cd frontend
npm install
npm run dev
